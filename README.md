# web-admin

Next.js admin dashboard for Kilat Pet Runner — users, bookings, payments, promos and reviews management.

## Stack

- Next.js 16 (App Router)
- React 19 + TypeScript 5
- Tailwind v4 + shadcn/ui (radix-ui primitives)
- react-hook-form + zod (forms + validation)
- zustand (client state)
- recharts (charts)
- axios (HTTP)

## Port

Runs on **`:3003`**.

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3003 to view the dashboard.

## Scripts

| Command | Purpose |
|---|---|
| `npm run dev` | Start the dev server on `:3003` |
| `npm run build` | Production build |
| `npm run start` | Run the production build |
| `npm run lint` | Lint the project |
